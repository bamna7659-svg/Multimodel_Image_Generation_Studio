# 🎨 Multimodal Image Generation Studio

An AI-powered web application built with **Python** and **Streamlit** that generates high-quality images from text prompts using Generative AI.

---

## ✨ Features

- 🖼️ AI Image Generation
- 🎨 Multiple Art Styles
- 🎲 Random Prompt Generator
- 📜 Image History
- 🖼️ Image Gallery
- 📥 Download Generated Images
- 🌙 Light & Dark Theme
- 🤖 AI FAQ Assistant
- 📖 How To Use Guide
- 👩‍💻 About Developer

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Hugging Face
- Groq API
- Generative AI

---

---

## 🚀 How to Run

1. Clone the repository.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Add your API keys in the `.env` file.

4. Run the application:

```bash
streamlit run app.py
```
